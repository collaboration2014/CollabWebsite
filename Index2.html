Project Overview

CORRECT WITH PAST TENSE Prince Edward County lies in Southern Ontario and has an unreliable form of water infrastructure data collection. The County currently uses paper, screenshots and emails for updating data which can lead to interpretations and inconsistencies in the data. The client of this project, Grant Hopkins, the GIS supervisor for the County, has asked for a fully digital way to enable the water department workers to collect and update water infrastructure data automatically and offline. This would be through an app with a customized user-friendly experience for all age demographics. Since this project will not have access to the County’s data and server, mock data and ArcGIS Online will be used. This project’s objectives involve creating a tool for the water department to collect/update data to ArcGIS Online automatically, the customization of the tool’s interface to be user-friendly and easy to use as well as ensuring data security. The beginning of this project will focus on the compatibility of Collector and the mock data in ArcGIS Online. Then, it will shift to the customization of the user experience and interface of Collector. The next step in the methodology is to connect the data collection tool to ArcGIS Online so that any new data collected is updated. The client has a small local government Enterprise license which gives them access to ESRI products and other software such as Geocortex, so it has been requested that this project uses ESRI software such as ArcGIS Collector. Maintenance of this project would involve keeping regular and new staff trained with any new updates that are made to the data collection software, the client configuring our product with their infrastructure and updating base maps used in the tool if they are no longer accurate. This project fulfils the client’s needs by eliminating paper data collection and providing a fully digital user-friendly experience to collect and update water infrastructure data
